# GitHub Pages - Grab Red Envelope

This directory contains the static website for Grab Red Envelope, hosted via GitHub Pages.

## Pages

- **index.html** - Homepage with app features and overview
- **privacy.html** - Complete privacy policy
- **support.html** - Support page with FAQs and contact information

## Enabling GitHub Pages

1. Go to your GitHub repository: https://github.com/imentos/GrabRedEnvelope
2. Click **Settings** (top menu)
3. Scroll down to **Pages** (left sidebar)
4. Under "Source", select:
   - **Branch:** main
   - **Folder:** /docs
5. Click **Save**
6. Wait 1-2 minutes for deployment

## Accessing Your Site

Once enabled, your site will be available at:
```
https://imentos.github.io/GrabRedEnvelope/
```

Individual pages:
- Homepage: https://imentos.github.io/GrabRedEnvelope/
- Privacy Policy: https://imentos.github.io/GrabRedEnvelope/privacy.html
- Support: https://imentos.github.io/GrabRedEnvelope/support.html

## Updating Content

To update the website:

1. Edit the HTML files in this directory
2. Commit and push changes to the `main` branch
3. GitHub Pages will automatically rebuild (takes 1-2 minutes)

## App Store Submission

Use these URLs in your App Store Connect submission:

- **Privacy Policy URL:** `https://imentos.github.io/GrabRedEnvelope/privacy.html`
- **Support URL:** `https://imentos.github.io/GrabRedEnvelope/support.html`
