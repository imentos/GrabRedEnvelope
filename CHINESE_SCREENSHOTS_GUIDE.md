## 🛠️ Tools & Fonts

### Recommended Tools:
1. **Canva** (Easiest - has templates)
   - Go to Canva.com
   - Custom Size: 1290 x 2796 px
   - Upload screenshot as background
   - Add Chinese text with PingFang font

2. **Figma** (Professional)
   - Free for personal use
   - Better control over gradients and shadows
   - Can export at 2x for crisp text

3. **Photoshop** (If you have it)
   - Most powerful
   - Best typography control

### Chinese Fonts:

**For Simplified Chinese (简体):**
- **Primary:** PingFang SC (苹方-简) - Apple's system font ✅ BEST
- Alternative: Noto Sans CJK SC, Source Han Sans SC
- ❌ Avoid: Microsoft YaHei (Windows look), SimSun (outdated)

**For Traditional Chinese (繁體):**
- **Primary:** PingFang TC (苹方-繁) - Apple's system font ✅ BEST
- Alternative: Noto Sans CJK TC, Source Han Sans TC
- ❌ Avoid: MingLiU (old), PMingLiU

**Font Weights:**
- Main Caption: PingFang SC/TC **Semibold** or **Bold**
- Subtitle: PingFang SC/TC **Regular** or **Medium**

### Font Sizes:

Chinese characters are more complex than English, so use slightly smaller sizes:

| Element | English | Chinese (SC/TC) |
|---------|---------|-----------------|
| Main Caption | 60-80pt | 50-70pt |
| Subtitle | 32-42pt | 28-38pt |

## 🎨 Design Guidelines

### Colors:
- **Text:** White (#FFFFFF) with drop shadow
- **Gradient Overlay:** Black to transparent, 60-70% opacity
- **Accent Colors:** Red (#FF0000) and Gold (#FFD700) for Chinese markets

### Layout:
```
┌─────────────────────┐
│                     │
│   主标题 (Top 20%)  │ ← Main Caption (White, Bold, 50-70pt)
│                     │
│                     │
│   [Screenshot]      │ ← Keep middle area clear (40-60%)
│                     │
│                     │
│   副标题 (Bottom 20%)│ ← Subtitle (White, Regular, 28-38pt)
│                     │
└─────────────────────┘
```

### Text Effects:
1. **Drop Shadow:** Offset 0, 0 / Blur 8px / Opacity 80% / Black
2. **Gradient Overlay:** 
   - Top: Black (#000000) 70% opacity → Transparent
   - Bottom: Black (#000000) 70% opacity → Transparent
3. **Alignment:** Center-aligned for impact
4. **Letter Spacing:** Normal or +1% (Chinese doesn't need much spacing)

## 📷 Step-by-Step: Creating Screenshots in Canva

### Step 1: Capture Game Screenshots
1. Run GrabRedEnvelope app on iPhone 15 Pro Max
2. Take screenshots of:
   - Screenshot 1: Gameplay with envelopes and players
   - Screenshot 2: Lobby screen with players joining
   - Screenshot 3: Active gameplay, multiple player cards visible
   - Screenshot 4: Beautiful red envelopes close-up
   - Screenshot 5: Winner celebration with fireworks
   - Screenshot 6: Results screen or clean lobby

### Step 2: Create Design in Canva

1. **Go to Canva.com** → "Custom Size"
2. **Enter dimensions:** 1290 x 2796 px
3. **Upload** your base screenshot as background
4. **Add Gradient Overlays:**
   - Top: Add rectangle, set to black gradient (top solid → bottom transparent)
   - Bottom: Add rectangle, set to black gradient (bottom solid → top transparent)
   - Set opacity to 60-70%

5. **Add Text:**
   - **Main Caption:**
     * Font: PingFang SC Semibold (for Simplified)
     * Size: 60pt
     * Color: White
     * Effect: Add drop shadow (blur 8px)
     * Position: Top 20% of image
     * Alignment: Center
   
   - **Subtitle:**
     * Font: PingFang SC Regular
     * Size: 35pt
     * Color: White or Light Gray (#E0E0E0)
     * Effect: Add drop shadow (blur 6px)
     * Position: Bottom 20% of image
     * Alignment: Center

6. **Download:**
   - Format: PNG
   - Quality: High
   - Resolution: 2x (for crisp text)

### Step 3: Create 5.5" Versions

1. Resize design to 1242 x 2208 px
2. Keep same captions (text will auto-scale)
3. Download as PNG

### Step 4: Repeat for All Versions

Create 3 complete sets:
- ✅ 6 screenshots × Simplified Chinese (简体)
- ✅ 6 screenshots × Traditional Chinese (繁體)
- ✅ 6 screenshots × English (already done)

Total: **18 unique screenshots** (6 captions × 3 languages)

For each language:
- 6 screenshots @ 1290 x 2796 px (6.7")
- 6 screenshots @ 1242 x 2208 px (5.5")

## 🆘 Troubleshooting

**Problem:** Chinese text looks pixelated
- **Solution:** Export at 2x resolution in Canva

**Problem:** Font doesn't look right
- **Solution:** Make sure you're using PingFang, not a Western font

**Problem:** Text is too small to read
- **Solution:** Keep main caption at least 50pt for Chinese

**Problem:** Gradient too dark, can't see screenshot
- **Solution:** Reduce gradient opacity to 60% or use thinner gradient bars

**Problem:** Traditional Chinese looks wrong
- **Solution:** Verify you're using TC (繁體) not SC (简体) characters
